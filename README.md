# Surgeon Speedrunner

A simple patch that adds QoL speedrunning changes for Surgeon Simulator.

---

## Features

### Improved Timer
- The in-game timer now shows milliseconds

### Fast Restart
- Press "F" to restart the level at any point. This is especially nice to get skip past the death screen.

---

## Installation
- Download the ```Assembly-CSharp.dll```
- Navigate to your game's root directory and put ```Assembly-CSharp.dll``` under ```ss2013_Data/Managed/``` depending on your OS:
  - Windows: ```C:\Program Files (x86)\Steam\steamapps\common\Surgeon Simulator\ss2013_Data\Managed\```
  - Linux: ```~/.local/share/Steam/steamapps/common/Surgeon Simulator/ss2013_Data/Managed/```
